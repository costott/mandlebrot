FRACTAL RENDERER

by costott

Preview
Features
  • Render the Mandelbrot and Julia sets (Buhddabrot coming soon)
  • Move around and zoom to explore the fractals as you please
  • Advanced menus, allowing you to completely customise your fractal images
  • Supports multiple layers to create more sophisticated images
  • Export and save both images and videos

Technical Details
  • Created in pure Rust with custom code (including the menus)
  • Source code: Github page
  • Can zoom up to 5e305x magnification (deeper coming soon), using arbirary precision numbers for deep zooms
  • Uses mandelbrot perturbation theory to dramatically improve render times for deep zooms
  • 11,000+ lines of code
